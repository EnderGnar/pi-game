# The Primary Game Mode
The game is very simple, please just implement an html file, I think we can even include the JS in it.
The idea of the game is to type out PI as quickly as possible. It should be playable both on mobile and with a keyboard.

## Visuals
The game should start with 3."input" in the center. where the input is exactly centered.
The numbers while we type should move to the left on some shape, my prefered option is to be able to give a function for
both position and scale while they run to the side. The movement between the positions should be snappy when a new number is added.
If a wrong number is typed, it should fall down somehow.

Initially make it about typing the first 50 digitis of pi, where we can see the next digits already. Then with the game mode it should be possible to toggle this visual.

Above the number is a timer.

Below is a keypad for mobile players.

Generally the vibe should be a bit glassy and rounded. Please make the background darker blue purple
and the numbers different neon flashy colors. Outlines I think white.

## Functionalities
Toggle for seeing the next number or not.
Selection for the length of the sequence.
Timer from when we start playing until we reached the end of our sequence.
